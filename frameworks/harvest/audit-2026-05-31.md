## Prioritized "harvest now" backlog

In dependency/leverage order — each is a follow-up PR:

1. **`frameworks/edge-proxy/`** — byte-identical in 3 spokes + a 5-invariant conformance suite; the most-proven, most-stable, fully-generic piece. Highest confidence.
2. **`frameworks/workflow-engines` promotion** — replace the stub README with WSC's proven schema+runner+checkpoint engine (already staged). Biggest real gap-fill.
3. **`frameworks/health-endpoint`** — the only true code pattern duplicated across all 7 edge workers.
4. **Hook hardening bundle** — circuit-breaker + timer wrappers, `rules/21-hooks/*`, and the 3 missing git guards (each born from a documented incident). Security the foundation currently lacks.
5. **Claims register → `frameworks/copywriting`** — the checker enforces the *idea* but lacks the *data model*.
6. **`frameworks/ui-patterns/` pointer** — documents the nav+footer contract and drives spokes off local copies onto `@wave-av/spoke-chassis`. **Unblocks #120/#121** (see below).
7. **consume.sh-managed shared files** — `.coderabbit.yaml` + `SECRETS.md`/`threat-model.md` templates as a single synced source.
